# 혜련의 웹페이지 (diding.github.io)

혜련의 개인 웹사이트로, 교육용 인터랙티브 게임과 도구를 제공합니다.

## 🌟 주요 기능

### 1. 메인 페이지 (index.html)
- **글래스모피즘 디자인**: 현대적인 반투명 유리 효과의 UI
- **리퀴드 애니메이션**: 유동적인 그라데이션 배경 블롭 효과
- **마우스 인터랙션**: 마우스 움직임에 반응하는 조명 및 3D 틸트 효과
- **네비게이션**: 각 기능으로 이동할 수 있는 직관적인 메뉴

### 2. 딥러닝 게임 (deeplearning.html)
AI와 딥러닝 개념을 게임으로 배우는 교육용 콘텐츠

**게임 방식:**
- **학습 단계**: AI에게 도형(동그라미, 네모, 세모, 별)과 이름을 가르치기
- **테스트 단계**: AI가 학습한 내용을 퀴즈로 확인
- **레벨 시스템**: 5단계의 난이도로 구성
- **포인트 & 배지**: 점수와 배지를 모아가며 진행

**주요 특징:**
- 로컬 스토리지를 활용한 최고 점수 저장
- 프로그레스 바로 학습 진행도 표시
- 애니메이션 피드백 (정답/오답 효과)
- 업적 시스템 (완벽한 정답 시 특별 배지 획득)

### 3. 자리 배치 시스템 (seatingarrangement.html)
학급 자리를 랜덤으로 배정하는 도구

**기능:**
- **학생 명단 관리**: 1~42명까지 학생 수 설정 가능
- **랜덤 이름 생성**: 한국 이름 자동 생성 기능
- **드래그 앤 드롭**: 책상을 자유롭게 배치 가능
- **자동 배정**: 랜덤 알고리즘으로 공정한 자리 배치
- **교실 레이아웃**: 유연한 그리드 배치 (6x6 그리드)

**사용 방법:**
1. 학생 수 입력 및 명단 생성
2. 책상을 원하는 위치로 드래그하여 배치
3. "자리 배정하기" 버튼으로 자동 배정

### 4. 영어 단어 게임 (englishword.html)
초등학생 수준의 영어 단어를 게임으로 배우는 교육용 콘텐츠

**게임 방식:**
- **행맨 게임**: 알파벳을 맞혀가며 숨겨진 단어 찾기
- **점수 시스템**: 정답 시 10점, 단어 완성 시 50점 추가
- **보상 시스템**: 게임을 계속할수록 보상(코인) 적립
- **힌트 제공**: 각 단어마다 한국어 힌트 제공

**주요 특징:**
- 초등학생 수준의 8개 단어 목록 (사과, 행복, 학교, 물 등)
- 기회 제한 시스템 (7번의 틀린 기회)
- 정답/오답에 따른 즉각적인 피드백
- 점수와 보상을 통한 게이미피케이션

## 🎨 디자인 특징

- **다크 모드 기반**: 눈의 피로를 줄이는 어두운 배경
- **그라데이션 효과**: 생동감 있는 컬러 그라데이션
- **반응형 디자인**: 모바일, 태블릿, 데스크톱 모두 지원
- **부드러운 애니메이션**: 자연스러운 전환 효과
- **인터랙티브 요소**: 마우스 hover 효과와 드래그 앤 드롭

## 🛠️ 기술 스택

- **HTML5**: 시맨틱 마크업
- **CSS3**: 글래스모피즘, 애니메이션, 그라데이션, 드래그 앤 드롭 스타일링
- **JavaScript (Vanilla)**: 게임 로직, DOM 조작, 드래그 앤 드롭, 상태 관리
- **Tailwind CSS**: 유틸리티 기반 스타일링 (index.html, seatingarrangement.html)
- **LocalStorage API**: 게임 데이터 및 점수 저장

## 🚀 사용 방법

### 로컬 실행
```bash
# 저장소 클론
git clone https://github.com/diding/diding.github.io.git

# 디렉토리 이동
cd diding.github.io

# index.html 파일을 브라우저로 열기
# 또는 간단한 HTTP 서버 실행
python -m http.server 8000
# 그 후 http://localhost:8000 접속
```

### GitHub Pages로 배포
- GitHub 저장소 Settings > Pages에서 활성화
- `https://diding.github.io` 에서 접속 가능

## 📋 게임별 상세 정보

### 딥러닝 게임 - 학습 메커니즘
- **1단계**: 사용자가 3가지 도형 예시를 선택하여 AI 학습 시뮬레이션
- **2단계**: AI가 학습한 도형의 카테고리를 구분하는 5가지 퀴즈 풀이
- **레벨별 진행**:
  - 레벨 1-3: 4개 도형 선택지
  - 레벨 4-5: 6-8개 도형 선택지 (난이도 상승)

### 자리 배치 시스템 - 배정 로직
- Fisher-Yates Shuffle 알고리즘으로 공정한 랜덤 배정
- 드래그 앤 드롭으로 자유로운 책상 배치 가능
- 책상 위치 교환(Swap) 지원

### 영어 단어 게임 - 난이도
- 초등학교 저학년 수준의 기본 영어 단어
- 각 단어별 한국어 힌트로 학습 보조
- 7번의 틀린 기회로 적절한 난이도 유지
